# 36个10天周期追踪器

一个Android应用，用于记录和管理每年36个10天周期的目标和结果。

## 项目概述

这个应用帮助用户：
1. 将一年划分为36个10天周期
2. 为每个10天周期设定目标
3. 记录每个周期的完成结果
4. 方便对照第1天和第10天的目标与结果

## 核心功能

### 1. 周期管理
- 自动计算每年的36个10天周期
- 显示当前周期和剩余天数
- 周期切换和导航

### 2. 目标设定
- 为每个周期设定具体目标
- 支持多个目标类别
- 目标进度追踪

### 3. 结果记录
- 记录每个周期的实际完成情况
- 对比目标与结果的差异
- 添加反思和笔记

### 4. 数据可视化
- 周期完成度统计
- 目标达成率图表
- 年度趋势分析

### 5. 提醒功能
- 周期开始/结束提醒
- 目标检查提醒
- 结果记录提醒

## 技术栈

- **前端**: Android (Kotlin + Jetpack Compose)
- **架构**: MVVM + Clean Architecture
- **数据库**: Room Database
- **状态管理**: ViewModel + StateFlow
- **依赖注入**: Hilt
- **测试**: JUnit, Espresso
